1. 此工具是使用powershell编写的windows的网线拔插监测工具
2. 执行方法为使用管理员权限开启powershell，然后cd到脚本所在目录，执行即可。
3. 执行时会提示输入一个要测试的网口可以ping通的IP
4. 测试开始前，请确保要测试的网口的网线是插入状态。测试过程中不要拔插其他网口，保持这些网口状态为开始测试时的状态。只拔插要测试的一个网口。
5.在测试过程中，再拔出网线之后，屏幕会提示让你重新插入；在插入之后，会查询网口的新的状态/IP/ping结果，等屏幕出现让重新拔出的提示时再拔出。
6.要结束测试，使用ctrl+c结束即可。
7.生成的结果在log\20180111111111_SIT_NETPLUG_TEST_WIN\20180111111111_SIT_NETPLUG_TEST_WIN.log文件中，详细记录了网口信息及每次测试结果。在结果文件中查找不到fail即为全部通过
